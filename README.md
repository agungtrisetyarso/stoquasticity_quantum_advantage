## Overview

This work provides a **structural, checkable criterion** (based on sign pattern and commutation structure) that determines when quantum thermal methods (Gibbs samplers and quantum annealers) can offer a genuine advantage over classical computation.

The repository includes:
- Python code to reproduce all numerical figures in the paper
- Implementations of the stoquasticity test and average sign calculation
- Code for the resonant coordination model and Davies generator gap analysis
- Data and notebooks for the advantage band analysis
